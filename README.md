# Калькулятор

Простой веб-калькулятор в стиле iPhone. Выполняет основные арифметические операции: сложение, вычитание, умножение и деление.

## Стек технологий

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** не требуется
- **База данных:** не требуется

## Ссылка на деплой

[Открыть калькулятор](https://s1lnce21.github.io/calculator-app/)

## Как пользоваться

1. Нажимайте на цифры (0-9)
2. Выберите действие (+, -, ×, ÷)
3. Введите второе число
4. Нажмите = для получения результата
5. Нажмите C для очистки

## Пример

`5 + 3 =` → `8`

## Особенности

- Последовательные вычисления (2 + 3 = 5, затем + 4 = 9)
- Защита от деления на ноль
- Подсветка активного оператора
